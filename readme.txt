Tweet your Woocommerce products to Twitter on publish/update automatically. Also shows live tweets on your site and add Tweet button in Woo products.

== Description ==

This is a simple plugin for your woocommerce site to upload/post your products to Twitter automatically. It will do the magic when you publish new product through woocommerce. It will be shared on your Twitter account. You can add Tweet button on products. It can also show your live tweets on your site by widget in any area.

= Pro features =

* Post products on Twitter
* Post product’s gallery images as gallery on Twitter
* post products with optional time delay between posting
* Post products multiple times (on every update)
* Bulk Tweets (multiple posts at once)
* Add Tweet button on product page
* Add live tweets widget
* Bulk Tweet button on/off option
* Rich product page
* And many more to come…

== Installation ==

To work properly, woocommerce should be installed before installing wootweet.

Here are few steps to follow:

1. Upload `wootweet` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select WooTweet Logins page from wordpress menu and login.
4. That's it. You're ready to go!

== Frequently Asked Questions ==

= How can I get App for Twitter? =

You can get from https://apps.twitter.com/.

= What if I don't want to tweet a product? =

If you uncheck the box 'Tweet it?' in publish area, it will not be tweeted.

= What if my product details exceeds 140 charcters? =

In this case, only link of your product will be published along with product image.
